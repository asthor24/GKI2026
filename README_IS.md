# GKI2026

GKI2026 er kóðasafn fyrir Íslandsmótið í gervigreind 2026. Það inniheldur þrjú AI/ML verkefni. Hver undirmappa inniheldur  verkefni, dæmi um líkanakóða og þær lágmarks-skrifstafir / skriftur sem þarf til að keyra eða staðfesta innsendingar.

Uppbygging (efst):
- `golden_plate_on_thingvellir_NLP/` — NLP-verkefni: spá um næsta bæti (gagna- og þjálfunartól, innsendingartól)
- `habitat_classification/` — myndaflokkunarverkefni með líkani og hjálparföllum
- `hierarchical_time_series_forecasting/` — verkefni í tímaraðarspám
- `submission/` eða verkefnissértækar `submission/` möppur — dæmi um `model.py` og innsendingarsniðmát

 Opnaðu verkefnismöppu, lestu `README_IS.md` (eða `README_IS.md` fyrir ensku) og fylgdu leiðbeiningunum.
